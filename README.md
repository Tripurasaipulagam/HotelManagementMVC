# Hotel Management MVC

**Project:** Hotel Management Services  
**Repository:** https://github.com/Tripurasaipulagam/HotelManagementMVC

## Overview  
This is a web-based Hotel Management application built using ASP.NET MVC. It enables key hotel operations such as room booking, customer management, check-in / check-out, and back-office functions in a structured MVC architecture.

## Features  
- Room inventory management (types, availability, rates)  
- Customer booking, check-in and check-out workflows  
- Staff / user authentication and role-based access (e.g., front desk, manager)  
- Reporting dashboards (occupancy, revenue, bookings by date)  
- Responsive UI built using HTML / JavaScript for front-end interactivity  
- Built with C# in ASP.NET MVC framework and backed by SQL Server for data persistence  
- Modular folder structure enabling easy extension (e.g., adding new modules like restaurant, billing, housekeeping)

## Technology Stack  
- .NET Framework (ASP.NET MVC)  
- C# for server-side logic  
- SQL Server (or any compatible RDBMS) for database  
- HTML, CSS, JavaScript (and optionally jQuery / Bootstrap) for front-end  
- Git & GitHub for version control
